## How to Turn Your AI Interns Into Middle Management

Let me tell you about the most beautifully ironic thing I've done all month.

I was sitting in Antigravity editor, running on my Mac, talking to GitHub Copilot—Microsoft's AI, paying Microsoft $10/month for the privilege.

And I thought: "I wish Copilot had a special agent for security audits."

But Copilot's custom agent feature? It exists. It's just... tedious to set up. You have to write YAML. You have to define tool permissions. You have to actually think about what you want the agent to do.

So I did what any reasonable developer would do.

I opened Claude Code.

Yes. Claude Code. Anthropic's other AI coding assistant. The one that's not Copilot. The one that costs $25/month and the one i use with Ollama on lots of the wonderful chinese models like kimi, qwen, deepseek, glm and others.

I told Claude Code: "Build me app that creates agents for GitHub Copilot."

And it did.

## Architecture of Chaos 

Let me explain why this is funny.

GitHub Copilot is Microsoft's AI. It runs in their cloud. It's deeply integrated into their ecosystem. It has "custom agents" now—specialized AI personas you can create.

Claude Code is Anthropic's AI. It runs in my terminal using the brain of Kimi 2.6 (for now) and is somehow better at complex reasoning than all the other models. It uses the claude "skills"—reusable workflows that feel like magic.

I used Anthropic's API  to build a custom agent for Microsoft's AI.

That's like asking a Ford mechanic to design the engine for your Tesla. It's like hiring a chef from a French restaurant to write the menu for an Italian bistro. It's like... you get the idea. It's weird. It's wrong. It worked perfectly.

## The Code Whisperer

The best part? Claude Code built the app for me in about 50 minutes. I just described what I wanted:

- code - ( streamlit and CLI code)
- linting
- unit tests 
- comments
- documentation 
- github actions 

the cost was 0,16 EUR for the whole process , which is insane compared to what you get.
